# Contributing to Project Memory

Thanks for your interest in contributing!

## Ways to Contribute

### Tool Integrations

Add support for other AI coding tools:
- Create `tool-extensions/<tool-name>/` directory
- Follow the Claude Code extension as a template
- Include setup instructions and config files

### Templates

Improve or add new templates:
- ADR templates
- Failure log formats
- Data quirk documentation
- Session log structures

### Documentation

- Fix typos and improve clarity
- Add examples and use cases
- Translate to other languages

### Core System

- Improve the prompt-patterns automation logic
- Enhance session tracking
- Add new documentation categories

## How to Submit

1. **Create an issue first** - describe what you want to change
2. Fork the repo
3. Create a branch (`git checkout -b feature/your-feature`)
4. Make your changes
5. Test with your AI coding tool
6. Submit a pull request referencing the issue

## Guidelines

- Keep it tool-agnostic where possible
- Document your changes
- Follow existing file naming conventions
- Test with at least one AI coding tool
- Reference the issue in your PR
